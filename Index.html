Basic HTML topic

  1. Boiler plate: typing HTML at first time, Element vs attribute, Block element vs Inline element, enter (br), comment ()
  2. Head HTML, Element used:
    1. Title to name the tab opened in browser
  3. Body HTML, Block Element used:
    1. Paragraph (p), Header (h1,h2,until h6), horizontal line (hr)
    2. ordered list (ol) same as numbering and Unordered list (ul) same as bullets
      1. Inside it use list (li) then
    3. insert image (img), add url to the image, and alternative if image url down
    4. anchor tag (a), to create hyperlink/hypertext to another website. Now we create website instead of just webpage
  4. Body HTML, Inline element used:
    1. Italic(i) versus Emphasis(em), Bold(b) versus Strong(strong)

Intermediate HTML

  1. Table (table), inside table is thead, tbody, and tfoot
    1. table is structured into: head (thead), body (tbody), and footer (tfoot). Reason is to isolate with one another, so it's easier to make different format/styles with one another
    2. Inside body: insert row (tr),then insert cell (td) to add cell. Insert row again for next row
    3. Table Attribute: cell spacing
